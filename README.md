# Adhyayan - AI Research Paper Analyzer

AI-powered tool for analyzing research papers with semantic search and citation extraction.

## Features
- PDF upload and analysis
- AI-powered Q&A
- Semantic similarity search
- Citation extraction
- Google OAuth authentication

## Tech Stack
- Python, Flask, Gradio
- LangChain, FAISS
- Groq API
- Sentence Transformers
